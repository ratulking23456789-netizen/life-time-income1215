
Online BD — Demo Web App
-----------------------

This is a simple demo single-page application for "Online BD" containing:

- Register / Login (username + mobile + password)
- Deposit packs: 100..1000 — each gives a bonus of BDT 500 on "I Have Paid" (demo)
- Profile page with balance, deposits and claim history
- LocalStorage based backend (no server). Suitable for demo and testing.

How to use:
1. Download and unzip.
2. Open `index.html` in a mobile browser or desktop browser.
3. Register an account, then use Deposit → Pay → I Have Paid to credit bonus (demo).
